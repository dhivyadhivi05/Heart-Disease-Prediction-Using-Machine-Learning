# Heart Disease Prediction Using Machine Learning

## Project Overview

This project focuses on analyzing healthcare data and predicting heart disease using Machine Learning techniques. The dataset contains various medical and lifestyle-related attributes such as age, blood pressure, cholesterol level, BMI, diabetes, smoking habits, stress level, sleep hours, and more.

The main goal of this project is to identify important health factors affecting heart disease and build a prediction model using Python.

The project demonstrates important Data Science and Machine Learning concepts such as:

* Data Cleaning
* Handling Missing Values
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Feature Engineering
* Machine Learning Model Building
* Model Evaluation
